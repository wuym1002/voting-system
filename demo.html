快速部署指南
准备工作：确保您有GitHub账号，如没有请先注册
第一步：上传代码到GitHub
  1. 在GitHub创建新仓库：voting-system
  2. 在项目目录执行： 
    ./upload_to_github.sh
第二步：Railway部署
  1. 访问 railway.app
  2. 使用GitHub登录
  3. 点击 "New Project" → "Deploy from GitHub repo"
  4. 选择 voting-system 仓库
第三步：添加数据库
  1. 在项目页面点击 "Add Service"
  2. 选择 "Database" → "MySQL"
  3. 等待数据库创建完成
完成！您会获得一个全球可访问的HTTPS链接
